## 安装

```bash
$ yarn add redux-agg
```

或者

```bash
$ npm i redux-agg
```

## 简介

`redux-agg` 是一种数据流管理模式，其本质是 [redux](https://github.com/reduxjs/redux) 辅助工具，它使您能够很方便的管理状态以及处理副作用。

- 支持 触发副作用。
- 支持 副作用中间件。
- 支持 插件功能。
